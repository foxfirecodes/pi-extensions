# pi-extensions

> monorepo for my [pi](https://pi.dev) extensions :3

## packages

- [`packages/pi-agents-local-md`](packages/pi-agents-local-md) — loads `AGENTS.local.md` files as additional pi context.
- [`packages/pi-alerts`](packages/pi-alerts) — rings the terminal bell when pi needs your input.
- [`packages/pi-stash`](packages/pi-stash) — stashes and restores the current prompt with a shortcut.

## development

```bash
pnpm install
pnpm test
```
